# JavaScript Flappy Bird
A simple minigame created in HTML Canvas and vanilla JavaScript as part of the AGH Web Development course.

The project is published on GitHub Pages and can be played [here](https://turek-jakub.github.io/js-flappy-bird/).

## How to Play

| Action | Control |
| :--- | :--- |
| **Start Game** | `Spacebar` |
| **Flap / Jump** | `Spacebar` |

Avoid Obstacles and move through the gaps between the pipes.

## Project Structure

```text
├── assets/
│   ├── Flappy Bird/     # Sprites
│   ├── UI/              # UI overlay elements
│   └── Sound Efects/    # Audio files
├── index.html           # Main canvas layout
├── style.css            # Game styling and Game Over menu
```
